# Template

This template can be used as a starter for new repositories.

## How to use

- [ ] Create your own repository from this template with the **Use this template** button. ([more info](https://help.github.com/en/articles/creating-a-repository-from-a-template))
- [ ] Enable Sponsorships feature in your repo settings ([more info](https://help.github.com/en/articles/displaying-a-sponsor-button-in-your-repository#displaying-a-sponsor-button-in-your-repository))
- [ ] Update or replace `LICENSE.md` with your own.
- [ ] Remove everything above [Title](#Title) from `README.md`
